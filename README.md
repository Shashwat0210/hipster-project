Laravel Product Management System

Overview:-

This project is a backend-focused Laravel application demonstrating:

- Multi-authentication system (Admin & Customer)
- Secure product management
- Large-scale product import using queues and chunk processing
- Real-time user presence using WebSockets
- Testable and maintainable architecture

The emphasis is on backend correctness, scalability, and clarity, not UI design.

Tech Stack
- Laravel 12
- PHP 8.2
- MySQL / SQLite (for testing)
- Laravel Reverb (WebSockets)
- Laravel Queues (Database driver)
- maatwebsite/excel (CSV / Excel import)
- PHPUnit (Testing)
